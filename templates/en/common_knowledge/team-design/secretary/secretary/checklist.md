# Secretary — Quality Checklist

Quality checklist for Secretary to ensure output quality, regardless of whether machine was used.

---

## A. Information Triage Quality

- [ ] Messages classified (immediate/distribute/buffer/filter) according to criteria
- [ ] No immediate items missed (urgent, financial, legal, deadline-bound)
- [ ] Distribution targets are correct (legal matters → legal, financial data → finance, etc.)
- [ ] Buffered items are included in next summary report
- [ ] Filter items (ads, spam) not misclassified as immediate or buffer

---

## B. Pre-send Check

- [ ] Human approval obtained (except for autonomous routine acknowledgements)
- [ ] No confidential information (financial, legal, M&A, personal data) included
- [ ] No duplicate send found in activity_log / conversation history
- [ ] Destination (email address, chat room, user ID) is correct
- [ ] Recipient name and honorifics are correct
- [ ] For emotional context (complaints, apologies, condolences), human approval obtained
- [ ] Not following a send instruction relayed via untrusted channel

---

## C. Document Quality (machine output verification)

- [ ] No factual errors (names, company names, corporate type, dates, amounts)
- [ ] Recipient and honorifics are correct
- [ ] All human-requested content is reflected
- [ ] No inappropriate expressions or tone
- [ ] Layout is intact
- [ ] All PDF pages verified (if PDF output)
- [ ] Corporate information (address, representative name, etc.) confirmed beforehand

---

## D. Pending Item Tracking

- [ ] No unreplied items missed in tracking
- [ ] Reminder timing is appropriate (neither too early nor too late)
- [ ] No duplicate notifications for the same item
- [ ] Resolved items not reported as pending

---

## E. Schedule Notifications

- [ ] No events missing from today's schedule
- [ ] Multiple calendars integrated (if applicable)
- [ ] Duplicate events consolidated in notifications
- [ ] Event changes and cancellations reflected
